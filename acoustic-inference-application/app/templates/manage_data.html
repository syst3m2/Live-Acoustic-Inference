Update Ship Classes

On this page you can select the correct class for 'Uknown' ship designations, or change the class of designations. The ship designations are currently treated as Unknown and discarded in the data. Please select the appropriate class they should be categorized into so future ships of this class can be recorded.

Please refer to the following research to familiarize yourself with the ship class cateorization convention: ShipsEar: An Underwater Vessel Noise Database

Changes made via this form will not reflect in past the True Labels used to evaluate model metrics, only future labels.
